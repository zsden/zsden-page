# Hello World

欢迎来到我的博客！这是使用 Elysia + Bun + Supabase 构建的个人博客系统。

## 技术栈

- **后端**: Elysia + Bun
- **数据库**: Supabase (PostgreSQL)
- **前端**: Vue 3 + TypeScript
- **内容**: Markdown

## 特性

- 支持Markdown文章编写
- 标签和分类管理
- 访问统计
- RSS订阅
- 响应式设计

## 代码示例

```javascript
const greet = (name) => {
  return `Hello, ${name}!`
}

console.log(greet('World'))
```

## 总结

这是一个现代化的博客系统，利用了最新的Web技术栈来提供优秀的用户体验。
