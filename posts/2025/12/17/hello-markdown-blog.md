# Hello Markdown Blog

欢迎来到我的基于 Markdown 文件的博客系统！

## 特性

这个博客系统具有以下特性：

- ✅ 基于 Markdown 文件
- ✅ 无需数据库
- ✅ 支持标签和分类
- ✅ 代码高亮
- ✅ 自动生成目录

## 代码示例

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet("World");
```

## 总结

这是一个简单而高效的博客解决方案。
